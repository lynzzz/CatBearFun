class MyCalendarTwo {

      List calendar = new ArrayList<>();
      List doubleBook =   new ArrayList<>();

      public MyCalendarTwo() {

      }

      public boolean book(int start, int end) {
          for( int[] interval : doubleBook){
              if ( start < interval[1] && end > interval[0]){
                  return false;
              }
          }

          for( int[] interval : calendar){
              if ( start < interval[1] && end > interval[0]){
                  doubleBook.add(new int[]{Math.max(interval[0], start), Math.min(interval[1],end)});
              }
          }

          calendar.add(new int[] {start, end});
          return true;
      }
  }

算法  保持一个二次重复的list. 当二次重复list在重复，就是第三次重复。 当一次重复list在重复，把公共部分加入二次重复。 每次值都加入一次重复

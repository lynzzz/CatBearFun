class Solution {
    public List> threeSum(int[] nums) {
        Arrays.sort(nums);
        List<List<Integer>> results = new ArrayList<>();
        
        
        for( int i=0; i<nums.length-2; i++){
 
            while( i>=1 && i<nums.length-2 && nums[i] == nums[i-1]){   外循环去重
                i++;
            }
            
            int j=i+1;
            int k=nums.length - 1;
            

            
            while( j < k){
            
                
                int sum = nums[i] + nums[j] + nums[k];
            
                if ( sum == 0){
                    List result = new ArrayList<>();
                    result.add(nums[i]);
                    result.add(nums[j]);
                    result.add(nums[k]);
                    
                    results.add(result);
                    
                    j++;
                    k--;
                    
                    while( j<k && nums[j] == nums[j-1] ){    //里循环去重 
                        j++;
                    }
                    
                    
                }else if ( sum < 0){
                    j++;
                }else{
                    k--;
                } 
           }
        }
        
        return results;
    }
}
1. 此题目的关键是两次去重。 里面循环去重，外面循环去重

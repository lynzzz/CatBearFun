public class Solution {
    public List binaryTreePaths(TreeNode root) {
        
		List ret = new ArrayList();
		if ( root == null){
		    return ret;
		}
		
		helper(root, ret, String.valueOf(root.val) );	
		return ret;
		
    }
    
    public void helper(TreeNode root, List results, String str){
	     
		 
		 if ( root.left == null && root.right == null ){
		     results.add( str );
			 return;
		 }
		 
		 if ( root.left != null ){
		     helper(root.left, results, str + "->" +  String.valueOf(root.left.val) );
		 }
		 
		 if ( root.right != null){
		     helper(root.right, results, str + "->" + String.valueOf(root.right.val));
		 }
	}
}
}
Time complexity : O(n)
Space complexity : Klg(n), where k is the number of path that meets the requirement
技巧 ： 1. 先把顶点放入小容器，好写一些， 2. String + 在recursion中不会带入其他递归层

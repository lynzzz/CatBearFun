Solution one : DFS 


    public int houseRobber3(TreeNode root) {
        // write your code here
        if ( root == null){
            return 0;
        }
        
        int sum = root.val;
        if ( root.left != null){
            sum = sum + houseRobber3(root.left.left) + houseRobber3(root.left.right);
        }
        
        if ( root.right != null){
            sum = sum + houseRobber3(root.right.left) + houseRobber3(root.right.right);
        }
        
        return Math.max(sum, houseRobber3(root.left) + houseRobber3(root.right));
        
    }

	
Time Complexity : O ( n * 2^n )， where n is the number of node in the tree



Solution two : DFS + DP (避免重复搜索)

 public int houseRobber3(TreeNode root) {
        // write your code here
        return helper(root, new HashMap());
    }
    
    public int helper(TreeNode root, HashMap m){
        
        if ( root == null){
            return 0;
        }
        
        if ( m.containsKey(root) ){
            return m.get(root);
        }
        
        int sum = root.val;
        if ( root.left != null){
            sum = sum  + helper(root.left.left, m) + helper(root.left.right, m);
        }
        
        if ( root.right != null){
            sum = sum + helper(root.right.left, m) + helper(root.right.right, m);
        }
        
        int ret = Math.max(sum, helper(root.left, m) + helper(root.right, m));
        m.put(root, ret);
        
        return ret;
    }

	时间复杂度降低，可以避免反复搜索

	Space : O(n)

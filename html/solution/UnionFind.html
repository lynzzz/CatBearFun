Union Find( disjoint set, 并查集）

应用：无向图分类找环
应用特征 图一般表达为 int[][] edges, 因为函数接口take两个顶点。 uf.union(A, B)
  • Valid Graph Tree(无向图找环：Union Find, BFS) Q
  • Number of connected components in an undirected graph（无向图分类)
  • Redundant Connection(无向图找环) Q
    • 算法： Step 1 : 图里的每一个点是一个独立的集
    step 2 : 按照你自己定义的法则进行合并。

    数据结构 : hashmap
    函数 ： constructor
    find: 返回这个节点的root
    union: 合并这两个节点的根节点, 每union(A, B)一次，和两个点就归为一类, 时间复杂度近似O(1) 
    


简单版本：
class UF
{
    int[] parent = new int[10000];

    UF(){
        for( int i=0; i<10000; i++){
            parent[i] = i;
        }
    }

    int find(int node){
        if ( parent[node] == node){
            return node;
        }
        return find(parent[node]);
    }

    void union(int n1, int n2){
        int p1 = find(n1);
        int p2 = find(n2);

        if ( p1 == p2){
          return;
        }
        parent[p1] = p2;
    }


}



复杂版本：带rank

class UF{

int[] parent;
int[] rank;

      UF(){
        for( int i=0; i<10000; i++){
            parent[i] = i;
            rank[i] = 0;
        }
	  }


	  private int find( int node ){
	       if ( parent[node] == node ){
		      return node;
		   }
	       return find( parent[node]);
	  }

	  public void union( int n1, int n2){

	    int p1 = find(n1);
			int p2 = find(n2);

      if ( p1 == p2 ) return;              // return if p1 and p2 are the same node.

			if ( rank[p1] > rank[p2] ) ){
			    parent[p2] = p1;
			}else if ( rank[p1] < rank[p2] ){
			    parent[p1] =p2;
			}else{
			    parent[p1]= p2;
			    rank[p2]++;
			}
	  }
}

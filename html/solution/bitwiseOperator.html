^ logic XOR operator: 0 if the same, 1 if different

a^a = 0
a^0 = a
a^b^c = a^c^b  (交换律)

2.
~ logic not operator:
a   1011
~a  0100

a & ~a = 0

3.
& logic and operator  1 if both are 1, otherwise 0
a&1 = a
a&0 = 0
a&~a = 0;

4.
| logic or operator: 1 if any of the two is 1. 0 if both are 0

1. 图的表示   

                       1  -  2  -  3
                             |
                             4

							 
a. adjacent matrix  ( 这种方法不是特别常见 ）           

       1  2   3  4 
     1    1  
     2 1      1  1
     3    1
     4    1
	 
	 
	 
b. Adjacent list    ( 用顶点来表示图,最常见的一种表达方式）
   1 - > { 2}
   2 - > { 1, 3 ,4 }
   3 - > { 2 }
   4 ->  { 2}
   
   表示方法一般为 
   class Node{
       public int label;
       List<Integer> neighbors;
   }
   
   List<Node> graph;
   
   或者
   HashMap<Integer, ArrayList<Integer>> Graph;     这种方法自己建图最好
   
   
   
c.   用边来表达图, 可以遍历转化成点图

int[][] graph = { {1, 2}, {2, 3}, {2,4}  } 边不会有重复表示 有了{1，2}就不会有{2，1}
建图的时候，要考虑到一条边有两条路径。 {1,2} => 表示的时候1->{2}, 2->{1}

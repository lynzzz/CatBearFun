1. 图的表示

                       1  -  2  -  3
                             |
                             4


a. adjacent matrix  ( 这种方法不是特别常见 ）

       1  2   3  4
     1    1
     2 1      1  1
     3    1
     4    1



b. Adjacent list    ( 用顶点来表示图,最常见的一种表达方式）
   1 - > { 2}
   2 - > { 1, 3 ,4 }
   3 - > { 2 }
   4 ->  { 2}

   表示方法一般为
   I：
   class Node{
       public int label;
       List<Node> neighbors;
   }

   List<Node> graph;

   或者II：
   HashMap<Integer, HashSet<Integer>> Graph; 这样可以保证图里没有重复的边
   如果写成    HashMap<Integer, List<Integer>> Graph; 这样图里可能会有重复的边。

   这两种图的表示方法有一个就可以了，不用互相转换
   第一种 n.neighbors 可以得到下一层
   第二种 graph.get(label) 可以得到下一层

c.   用边来表达图, 可以遍历转化成点图(Union Find特征，每条边的两个顶点依次Union)

int[][] graph = { {1, 2}, {2, 3}, {2,4}  } 边不会有重复表示 有了{1，2}就不会有{2，1}
建图的时候，要考虑到一条边有两条路径。 {1,2} => 表示的时候1->{2}, 2->{1}

1. 边数  == 点数 - 1  ： 连通图无欢条件

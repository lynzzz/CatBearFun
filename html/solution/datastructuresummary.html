Priority Queue

1. 如何初始化
PriorityQueue pq = new PriorityQueue<>() 默认的情况是min priority queue
priorityQueue pq = new PriorityQueue<>(k, Collections.reverseOrder()); 这样是max priority queue

2. 时间复杂度：
add/remove O(lgn)
min/max O(1)

3. 算法：
3.1 求第k largest number, 用min priority queue, 比头大就去掉头，往里加， 最后头是第k个最大的数。 记忆方法是，你如果要求数组最大值，一般初始化成Integer.MIN_VALUE
3.2 求第k smallest number 用max priority queue，比头小就去掉头，往里加， 最后头是第K个最小的数。 记忆方法是，你如果要求数组最小值，一般初始化成Integer.MAX_VALUE

4. 模板 ： 以求第k个最大的数字为例子

PriorityQueue pq = new PriorityQueue();

for( int i=0; i<num.length; i++){
     if ( pq.size() < k ){
	     pq.add(nums[i];
	 }else{
	     if ( nums[i] > pq.peek()){
		     pq.remove();
			 pq.add(nums[i];
		 }
	 }

}
time complexity : O( nlgk)
space : o(k);

public class Solution {
    /**
     * @param stack an integer stack
     * @return void
     */
    public void stackSorting(Stack stack) {
        // Write your code here
        Stack sortStack = new Stack<>();
        
        while( stack.size() != 0 ){
		    int cur = stack.pop();
            if ( sortStack.size() == 0 || cur >= sortStack.peek() ){
                sortStack.push(cur);
            }else{
                while( sortStack.size() != 0 && cur < sortStack.peek() ){
                    stack.push(sortStack.pop());
                }
				sortStack.push(cur);
            }
        }
        
        stack.addAll(sortStack);
    }
}
Time complexity : O(n^2)
Space complexity : O(n)
// Note
1. Maintain a sorted stack and a cur element
2. If cur > sortedStack.peek() 压进去， 否则出来一些在压进去

public class Solution {
    public ListNode reverseBetween(ListNode head, int m, int n) {
        if (m >= n || head == null) {
            return head;
        }
        
        ListNode dummy = new ListNode(0);
        dummy.next = head;
        head = dummy;
        
        for (int i = 1; i < m; i++) {

            head = head.next;
        }
        
        ListNode premNode = head;
        ListNode pre =null, cur = head.next;
        for (int i = m; i < n+1; i++) {
 
            ListNode temp = cur.next;
            cur.next = pre;
            pre = cur;
            cur = temp;
        }
        
        head.next.next = cur;
        head.next = pre;
        
        return dummy.next;
    }
}
Time O(n)
Note : 反转链表模板
1. 找到返转莲表前面那个点
2. 用模板反转链表
3. 在链回去

public class Solution {
    /**
     * @param s: A string
     * @return: A list of lists of string
     */
    public List> partition(String s) {
        // write your code here
        List> results = new ArrayList<>();
        dfs(results, new ArrayList(), 0, s);
        
        return results;
    }
    
    
    
    private void dfs(List> results, List result, int pos, String s){
        
        if ( pos == s.length() ){
            results.add( new ArrayList(result) );
            return;
        }
        
        
        for( int i= pos; i< s.length(); i++ ){
            String str = s.substring(pos, i+1);
            
            if ( !isPalindrome(str) ){
                continue;
            }
            
            result.add(str);
            dfs( results, result, i+1, s);
            result.remove(result.size()-1);
        }
    }
    
    
    private boolean isPalindrome( String s ){
        int start = 0;
        int end = s.length()-1;
        
        while( start < end){
            if ( s.charAt(start) != s.charAt(end) ){
                return false;
            }
            
            start++;
            end--;
        }
        
        return true;
    }
}
Note:
1. return all possible solutions -> DFS -> 观察答案发现无回溯 [a a b] [aa b] ->必定是subset模板
2. 领会此题目的精华， pos在每层位置是不变的，变的是i, 所以用pos做开头，i做结尾，可以在每层组成不同的substring

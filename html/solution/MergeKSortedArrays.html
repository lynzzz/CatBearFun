public class Solution {
    /**
     * @param arrays k sorted integer arrays
     * @return a sorted array
     */
    public List<Integer> mergekSortedArrays(int[][] arrays) {
        // Write your code here
            List results = new ArrayList<>();
            int[] ret = helper(arrays, 0, arrays.length - 1);
            
            for( int n : ret){
                results.add(n);
            }
            return results;
    }
    
    int[] helper(int[][] arrays, int start, int end){
        
        // return condition
        if ( start == end){
            return arrays[start];
        }
        
        
        //divide 
        int mid = start + ( end - start)/2;
        
        int[] l = helper(arrays, start, mid);
        int[] r = helper(arrays, mid + 1, end);
        
        // conquer
        return mergeSortedArray(l ,r);
    }   
    
        
    public int[] mergeSortedArray(int[] A, int[] B) {
        // Write your code here
        int[] ret = new int[A.length + B.length];
        
        int i=0, j=0, k=0;   
        
        while( i < A.length && j < B.length){
            if ( A[i] < B[j]){
                ret[k] = A[i];
                i++;
                k++;
            }else{
                ret[k] = B[j];
                j++;
                k++;
            }
        }
        
        while( i < A.length){
            ret[k] = A[i];
            i++;
            k++;
        }
        
        while ( j < B.length){
            ret[k] = B[j];
            j++;
            k++;
        }
        
        return ret;
    }
}
Time O(kn) , where n is the average length of each array. 相当于全部数字过一遍
Space O(1) , 不考虑递归的栈空间

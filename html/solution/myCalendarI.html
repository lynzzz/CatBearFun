class MyCalendar {
      TreeMap m = new TreeMap<>();

      public MyCalendar() {

      }

      public boolean book(int start, int end) {
          Integer preStart = m.floorKey(start);
          Integer nextStart = m.ceilingKey(start);

          if ( ( preStart == null || m.get(preStart) <= start) &&
               ( nextStart == null || end <= nextStart )){

              m.put(start, end);
              return true;
          }

          return false;
      }
  }


TreeMap sorted map, based on red-black tree, search is Olg(n)
TreeMap.floorKey(key)  returns the key that is less than or equal to key, returns null if not found
TreeMap.ceiling(key)  return the key that is greater than or equal to key, returns null if not found
